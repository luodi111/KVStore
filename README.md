# KVStore - 轻量级键值存储服务器
基于 C++ 实现的高性能键值数据库，支持网络通信、LRU 缓存、多线程并发、数据持久化。
## 功能特性
- **网络通信**：基于 TCP 协议，支持多客户端同时访问
- **LRU 缓存**：自动淘汰不常用数据，提升热点数据访问速度
- **线程安全**：使用 std::mutex 保证并发安全
- **数据持久化**：文件存储，程序重启后数据不丢失
- **配置文件**：支持自定义端口、缓存大小、数据文件路径
- **监控统计**：实时查看缓存命中率、总数据量、命令执行次数

### 编译
